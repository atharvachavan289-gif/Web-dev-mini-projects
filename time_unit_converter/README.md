# ⏳ Time Unit Converter

A fast and efficient web utility that allows users to seamlessly convert time values across multiple units. Whether you need to calculate seconds into hours, or days into minutes, this tool provides instant and accurate results with a clean user interface.

This project is part of the `Web-dev-mini-projects` collection.

## ✨ Key Features
* **Multi-Unit Support:** Easily convert between seconds, minutes, hours, days, and more.
* **Instant Calculation:** Performs the mathematical conversion in real-time as you interact with the inputs.
* **Intuitive UI:** Simple dropdown menus and input fields designed for a frictionless user experience.
* **No Dependencies:** Built entirely with pure web technologies for maximum performance.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for forms and input fields.
* **CSS3:** Custom styling for a responsive and clean layout.
* **Vanilla JavaScript:** DOM manipulation and mathematical conversion logic.
