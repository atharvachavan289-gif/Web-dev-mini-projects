# 💰 Expense Tracker

A clean, interactive web application designed to help users easily track their daily income and expenses. By visualizing your cash flow, this tool allows you to maintain a clear picture of your personal budget and financial health.

This project is part of the `Web-dev-mini-projects` collection.

## ✨ Key Features
* **Real-Time Balance:** Automatically calculates your total balance, income, and expenses as new data is entered.
* **Transaction History:** Displays a detailed, dynamic list of all past transactions.
* **Quick Edits:** Easily add new income/expense entries or delete incorrect ones with a single click.
* **Visual Indicators:** Color-coded UI components (green for income, red for expenses) provide immediate visual feedback.

## 🛠️ Tech Stack
* **HTML / JSX:** Semantic structure and component layout.
* **CSS3:** Responsive styling for a clean, distraction-free user interface.
* **JavaScript:** Core logic for state management, calculations, and DOM updates.
