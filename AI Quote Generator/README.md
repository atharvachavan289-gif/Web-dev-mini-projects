# 🤖 AI Quote Generator

A dynamic, interactive web application that fetches and displays inspiring, thought-provoking quotes using a third-party API. With a single click, users can generate fresh quotes instantly without reloading the page.

This project is part of the `Web-dev-mini-projects` collection.

## ✨ Key Features
* **Dynamic API Fetching:** Asynchronously retrieves quote data from an external API in real-time.
* **Instant Generation:** Delivers a seamless user experience by updating the UI instantly without page reloads.
* **Clean & Modern UI:** A minimalist, focused design that highlights the generated content.
* **Responsive Layout:** Perfectly scales across desktop, tablet, and mobile devices.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and content placeholders.
* **CSS3:** Modern styling, layout centering, and responsive design.
* **Vanilla JavaScript:** Asynchronous HTTP requests (Fetch API), JSON parsing, and DOM manipulation.
