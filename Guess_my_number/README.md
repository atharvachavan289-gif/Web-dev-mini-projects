🔢 Guess My Number


A fun, interactive web-based browser game where the player must guess a secretly generated number between 1 and 20. The game features dynamic score tracking, helpful hints, and saves your highest score during the session!

✨ Features
Interactive Gameplay: Guess the correct number with helpful hints provided (e.g., "📈 Too high!" or "📉 Too low!").

Dynamic Scoring: Start with a maximum score of 20, which decreases with every incorrect guess.

Highscore Tracking: Keeps a record of your best (highest) score across multiple rounds.

Play Again: Easily reset the game to play another round without losing your high score.

🛠️ Technologies Used
HTML5: For structuring the game interface and layout.

CSS3: For styling and visual feedback (like changing the background color when the player wins).

Vanilla JavaScript: For implementing the game logic, state management, and DOM manipulation.

🚀 Getting Started
Follow these simple steps to run the game locally on your machine.

Installation & Usage
Clone the repository:

Bash
git clone https://github.com/Ayushparikh-code/Web-dev-mini-projects.git
Navigate to the project directory:

Bash
cd Web-dev-mini-projects/Guess_my_number
Run the game:
Simply open the index.html file in your preferred web browser. You can do this by double-clicking the file or right-clicking and selecting "Open with -> [Your Browser]".

Play! 😉
Start guessing numbers between 1 and 20, watch your score, and try to set a new high score.
