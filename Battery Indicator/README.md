🔋 Battery Indicator App
A sleek, lightweight web application that displays the real-time battery percentage and charging status of your device. Built using the modern JavaScript Battery Status API, this tool runs entirely in your web browser with no external dependencies required.

✨ Features
Real-Time Percentage: Accurately displays your current system battery level.

Charging Status: Indicates whether the device is currently plugged in or running on battery power.

Responsive Design: Clean and accessible user interface.

Lightweight: Pure vanilla front-end code with zero external libraries.

🛠️ Technologies Used
HTML5: For structuring the application.

CSS3: For styling and visual indicators.

Vanilla JavaScript: For DOM manipulation and accessing the Battery Status API.

🚀 Getting Started
Follow these steps to run the project locally on your machine.

Prerequisites
A modern web browser (Chrome, Edge, Opera, etc.) that supports the Battery Status API.

Installation & Usage
Clone the repository:

Bash
git clone https://github.com/Ayushparikh-code/Web-dev-mini-projects.git
Navigate to the project directory:
(Note: Adjust the folder name below to match the exact folder of this app within the repository)

Bash
cd Web-dev-mini-projects/Battery-Indicator
Run the app:
Simply open the index.html file in your preferred web browser. You can do this by double-clicking the file or right-clicking and selecting "Open with -> [Your Browser]".
