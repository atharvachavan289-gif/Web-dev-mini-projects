# 🎲 2048 Game

A fun, interactive web-based clone of the classic 2048 puzzle game! Slide tiles, combine matching numbers, and test your logic to reach the elusive 2048 tile. 

This project is part of the `Web-dev-mini-projects` collection.

## ✨ Key Features
* **Classic Gameplay:** Authentic 2048 sliding mechanics and tile merging.
* **Keyboard Controls:** Fully responsive to arrow key inputs for a seamless playing experience.
* **Clean UI:** A minimalist design focusing entirely on the gameplay logic.

## 🎮 How to Play
* Use your **Arrow Keys** (Up, Down, Left, Right) to slide all tiles in that direction.
* When two tiles with the same number touch, they merge into one!
* Keep combining tiles to reach **2048** and win the game.

## 🛠️ Tech Stack
* **HTML5:** Game board structure and layout.
* **CSS3:** Grid styling, tile colors, and animations.
* **Vanilla JavaScript:** Game matrix logic, state management, and keyboard event listeners.
