# 🔍 GitHub Profile Finder

A lightweight web application that instantly fetches and displays detailed GitHub user profiles using the official GitHub REST API. Just enter a username to generate a clean profile card with live stats and a direct link to their repository.

## ✨ Key Features
* **Instant Search:** Fetches real-time profile data dynamically.
* **Detailed Profile Card:** Displays the user's profile picture, full name, and bio.
* **Live Stats:** Tracks the number of Public Repositories, Followers, and Following.
* **Direct Navigation:** Includes a clickable username that redirects straight to the user's official GitHub page.
* **Asynchronous Data:** Built using JavaScript Promises to ensure smooth, non-blocking UI updates while fetching data.

## 🛠️ Tech Stack
* **HTML5 & CSS3:** For structuring and styling the user interface.
* **Vanilla JavaScript:** For DOM manipulation and event handling.
* **Fetch API (Promises):** To handle asynchronous HTTP requests to the external API.

## 📡 API Reference
This project relies on the official public GitHub REST API to fetch user details.
* **Endpoint Used:** `https://api.github.com/users/{username}`

## 🏃‍♂️ How to Run Locally

Since this project uses pure web technologies, running it locally is incredibly simple:

1. **Clone or download the repository.**
2. **Navigate to the project folder.**
3. **Open the HTML file:**
   Simply double-click on `index.html` to open it in your default web browser. 
   *(Alternatively, if you are using VS Code, you can use the "Live Server" extension for a better development experience).*
